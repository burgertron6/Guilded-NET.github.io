What is Guilded.NET?

Guilded.NET is an API wrapper for Guilded. Guilded.NET allows you to interact with API easily. It uses .NET, so be sure to install it before using it.

It's currently using undocumented and unsupported API. Guilded developers will not help you in any way, shape or form.
